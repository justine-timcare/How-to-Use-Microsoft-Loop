How to Use Microsoft Loop

A beginner's guide to collaborative workspaces, pages, and components

Understand the basics

3 building blocks of Microsoft Loop

Workspaces

  • Shared spaces for your team or project
  • Group all related pages, files, and tasks together
  • Everyone can see progress and stay aligned

Pages

  • Flexible canvases for notes, plans, and ideas
  • Add text, tables, checklists, and components
  • Share as a link or embed in Teams and Outlook

Components

  • Portable content blocks (lists, tables, notes)
  • Stay in sync across Teams, Outlook, and Loop
  • Edit anywhere, updates show everywhere
The key idea: Loop components stay in sync everywhere. Edit a table in Teams, and it updates in Outlook and Loop automatically.

Get started

Step-by-step: create your first workspace

1

Go to Loop

Open your browser and go to loop.microsoft.com. Sign in with your work or personal Microsoft account.

2

Create a workspace

Click the "+" button or "Create new workspace." Give it a name (e.g., "Q3 Project Plan") and pick an icon and cover image.

3

Add team members

Click the share button in your workspace and invite people by name or email. They can view and edit immediately.

4

Create your first page

Click "+" to add a new page. Start from a blank page or use a template (meeting notes, project tracker, brainstorm, etc.).

5

Add content

Type "/" to see all the things you can insert: tables, task lists, voting tables, progress trackers, headings, dividers, and more.

Quick tip: Type / anywhere on a page to open the insert menu. This is the fastest way to add components.
Use it everywhere

Loop components in Teams and Outlook

In Microsoft Teams

  • In any chat or channel, click the Loop icon in the message box
  • Choose a component type (table, checklist, task list, etc.)
  • Everyone in the chat can edit it in real time
  • Changes sync back to Loop automatically

In Outlook

  • When composing an email, click the Loop component icon
  • Insert a table, checklist, or other component
  • Recipients can edit the component directly in the email
  • No more "which version is the latest?" confusion
This is the magic of Loop: one component, many locations, always in sync. Update a task in Teams and it instantly updates in Loop and Outlook.
Work smarter

Best practices for using Loop

Use templates to save time

Loop has built-in templates for meeting notes, project plans, brainstorms, and more. Start from a template instead of a blank page.

Turn on notifications

Follow workspaces and pages to get notified when teammates make changes. Click the bell icon on any page to follow it.

Use @mentions for task assignment

Type @ followed by someone's name to assign them a task or draw their attention. They will get a notification.

Organize with sections

Use headings and dividers to break up long pages. Create separate pages within a workspace for different topics.

Note: Loop workspaces and components count toward your Microsoft cloud storage. Keep an eye on your storage if you create many large workspaces.

Real-world use

Popular ways to use Loop

Meeting notes and action items

Use the Meeting Notes template to capture agendas, decisions, and action items. Share the Loop page in a Teams meeting so everyone can edit in real time.

Project tracking

Create a workspace for each project. Use task lists and progress trackers to keep everyone aligned. Assign tasks with @mentions.

Team brainstorming

Use a blank page or the Brainstorm template to collect ideas from the team. Everyone can add, vote, and comment in real time.

Shared checklists

Create a checklist in Loop and paste it as a component in a Teams channel. As people check items off, everyone sees the progress instantly.

Status updates via email

Insert a Loop table in an Outlook email for weekly status updates. Recipients can fill in their updates directly in the email without opening Loop.

Fix it fast

Troubleshooting common issues

I cannot access Loop

Make sure you are signed in at loop.microsoft.com with your work or personal Microsoft account. Your admin may need to enable Loop for your organization.

Loop components are not showing in Teams

Update your Teams app to the latest version. Loop components require the new Teams experience. Classic Teams may not support them.

Changes are not syncing

Check your internet connection. Loop requires an active connection to sync in real time. Try refreshing the page.

Cannot share a workspace with external users

External sharing is limited. Your IT admin controls whether Loop can be shared outside your organization. Check with your admin.

Loop is running slowly

Large pages with many components can slow down. Break up content into multiple pages within the same workspace.
